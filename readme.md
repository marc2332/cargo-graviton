## Developing
Clone the repo to the desired .graviton2/plugins:
```shell
git clone https://github.com/marc2332/cargo-graviton.git 
```

Install dependencies:
```shell
npm install
```

Run locally:
```shell
npm run watch
```

Build (optional):
```shell
npm run build
```
